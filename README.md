# Plataforma de Gestão Escolar

Este projecto tem como objectivo criar uma plataforma simples para gestão de escolas do ensino secundário em Angola.  
A aplicação é desenvolvida em **Python (Flask)** com uma base de dados **SQLite** e um frontend em **HTML, CSS e JavaScript**.
## Funcionalidades Principais
- Gestão de alunos  
- Gestão de professores  
- Gestão de turmas e disciplinas  
- Registo de notas e avaliações  
- Autenticação de utilizadores (admin, professor, secretário)  
- Painel de administração

## Tecnologias Utilizadas
- **Backend:** Python + Flask  
- **Base de Dados:** SQLite  
- **Frontend:** HTML, CSS, JavaScript  
- **Outras:** Jinja2, Bootstrap (opcional)
